Task L10

Create a function that will create hobbits. That is, return an object that we will consider a hobbit.

Hobbit - object. which has fields. Name, age and counter.

The name must be passed to the function. If no name is given DO NOT create a hobbit.

Age (integer) should be generated automatically and randomly in the range from 12 to 100. We will assume that years.

The counter is also generated automatically. And it carries a meaning - how the hobbit was created in a row.

Then use this function to create 24 hobbits with different names. Output to the console as an array.

P.S. To complete the task, of course, create an html page. It is desirable to connect scripts through the "src" attribute.
