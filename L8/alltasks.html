Sorting arrays

  1. Given two one-dimensional arrays with different numbers of elements and a natural number k. Merge them into one array, including the second array between k-th and (k+1)-th elements of the first, without using an additional array.
  2. Given two sequences a1<=a2<=...<=an and b1<=b2<=...<=bm. Form from them a new sequence of numbers so that it is also non-decreasing. Do not use an additional array.
  3. Selection sort. Given a sequence of numbers a1,a2,...an. Need to rearrange elements so that they are in descending order. To do this, in an array, starting with first, the largest element is selected and placed in first place, and the first - in the place of the greatest. Then, starting from the second, this procedure is repeated. Write selection sort algorithm.
  4. Exchange sorting. Given a sequence of numbers a1, a2,...an. Need to rearrange numbers in ascending order. To do this, two adjacent numbers ai and ai+1 are compared. If ai>ai+1, then a permutation is made. This continues until all elements are will be in ascending order. Write a sorting algorithm while counting the number of permutations.
  5. Insertion sort. Given a sequence of numbers a1, a2,...an. Required rearrange the numbers in ascending order. This is done in the following way. Let a1,a2,...,ai is an ordered sequence, i.e. a1<=a2<=...<=ai. taken the next number ai+1 and is inserted into the sequence so that the new the sequence was also increasing. The process continues until all elements from i+1 to n will not be iterated over.
